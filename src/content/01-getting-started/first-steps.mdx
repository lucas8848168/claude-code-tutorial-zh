# 第一步：创建你的第一个项目

现在你已经安装好了 Claude Code，让我们通过一个简单的示例来体验它的强大功能。

## 创建新项目

### 1. 打开 Claude Code

启动 Claude Code 应用程序，你会看到欢迎界面。

### 2. 创建项目文件夹

有两种方式创建新项目：

**方式一：通过界面**
1. 点击"文件" > "打开文件夹"
2. 选择或创建一个新文件夹（例如 `my-first-project`）
3. 点击"选择文件夹"

**方式二：通过命令行**
```bash
# 创建项目文件夹
mkdir my-first-project
cd my-first-project

# 使用 Claude Code 打开
claude .
```

## 第一个 Python 示例

让我们创建一个简单的 Python 程序，体验 Claude Code 的代码补全和 AI 助手功能。

### 1. 创建 Python 文件

1. 在左侧文件浏览器中，点击"新建文件"图标
2. 输入文件名：`hello.py`
3. 按 Enter 键创建文件

### 2. 使用代码补全

在 `hello.py` 文件中，开始输入以下代码：

```python
# 这是我的第一个 Python 程序
def greet(name):
    """向用户打招呼"""
    message = f"你好，{name}！欢迎使用 Claude Code。"
    return message

# 主程序
if __name__ == "__main__":
    user_name = input("请输入你的名字：")
    greeting = greet(user_name)
    print(greeting)
```

**体验智能补全：**
- 当你输入 `def` 时，Claude Code 会自动建议函数定义
- 输入 `f"` 时，会提示 f-string 格式
- 输入 `if __name__` 时，会自动补全完整的主程序模板

### 3. 运行程序

在终端中运行你的第一个程序：

```bash
python hello.py
```

输入你的名字，程序会向你打招呼！

## 使用 AI 助手

Claude Code 最强大的功能之一是 AI 助手。让我们用它来改进我们的程序。

### 1. 打开 Chat 面板

- 点击左侧的聊天图标，或
- 使用快捷键 `Ctrl+L`（Windows/Linux）或 `Cmd+L`（macOS）

### 2. 与 AI 对话

在聊天框中输入：

```
请帮我改进这个程序，添加以下功能：
1. 支持多种语言的问候（中文、英文、日文）
2. 添加错误处理
3. 让代码更加优雅
```

AI 助手会分析你的代码，并提供改进建议。你可以直接应用这些建议，或者继续讨论。

### 3. 应用建议

AI 可能会建议类似这样的改进代码：

```python
# 改进后的多语言问候程序
def greet(name, language="zh"):
    """
    向用户打招呼
    
    参数:
        name (str): 用户名字
        language (str): 语言代码 ('zh', 'en', 'ja')
    
    返回:
        str: 问候消息
    """
    greetings = {
        "zh": f"你好，{name}！欢迎使用 Claude Code。",
        "en": f"Hello, {name}! Welcome to Claude Code.",
        "ja": f"こんにちは、{name}さん！Claude Code へようこそ。"
    }
    
    return greetings.get(language, greetings["zh"])

def get_user_input(prompt):
    """安全地获取用户输入"""
    try:
        return input(prompt).strip()
    except (KeyboardInterrupt, EOFError):
        print("\n程序已退出。")
        return None

def main():
    """主程序"""
    print("=== 多语言问候程序 ===")
    
    # 获取用户名
    user_name = get_user_input("请输入你的名字：")
    if not user_name:
        return
    
    # 选择语言
    print("\n选择语言：")
    print("1. 中文 (zh)")
    print("2. English (en)")
    print("3. 日本語 (ja)")
    
    lang_choice = get_user_input("请输入选项 (1-3)：")
    
    # 语言映射
    lang_map = {"1": "zh", "2": "en", "3": "ja"}
    language = lang_map.get(lang_choice, "zh")
    
    # 显示问候
    greeting = greet(user_name, language)
    print(f"\n{greeting}")

if __name__ == "__main__":
    main()
```

点击代码块旁边的"应用"按钮，AI 会自动更新你的文件。
