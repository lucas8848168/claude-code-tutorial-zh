# 安装 Claude Code

本指南将帮助你在不同操作系统上安装 Claude Code，并完成基本配置。

## 系统要求

在开始安装之前，请确保你的系统满足以下要求：

- **操作系统**：Windows 10/11、macOS 10.15+、或主流 Linux 发行版
- **内存**：至少 4GB RAM（推荐 8GB 或更多）
- **磁盘空间**：至少 500MB 可用空间
- **网络连接**：需要互联网连接以下载和激活

## Windows 安装步骤

### 1. 下载安装程序

访问 [Claude Code 官方网站](https://code.claude.com)，点击"下载"按钮，选择 Windows 版本。

### 2. 运行安装程序

1. 双击下载的 `.exe` 文件
2. 如果出现安全提示，点击"运行"
3. 选择安装路径（默认为 `C:\Program Files\Claude Code`）
4. 点击"安装"按钮

### 3. 完成安装

安装完成后，你可以：
- 从开始菜单启动 Claude Code
- 在桌面上找到快捷方式
- 右键点击文件夹选择"使用 Claude Code 打开"

### Windows 常见问题

**问题：安装时提示"需要管理员权限"**
- 解决方案：右键点击安装程序，选择"以管理员身份运行"

**问题：防火墙阻止连接**
- 解决方案：在 Windows 防火墙设置中允许 Claude Code 访问网络

## macOS 安装步骤

### 1. 下载安装包

访问官方网站下载 macOS 版本（`.dmg` 文件）。

### 2. 安装应用

1. 双击下载的 `.dmg` 文件
2. 将 Claude Code 图标拖动到"应用程序"文件夹
3. 等待复制完成

### 3. 首次启动

1. 打开"应用程序"文件夹
2. 双击 Claude Code 图标
3. 如果出现"无法打开，因为它来自身份不明的开发者"提示：
   - 打开"系统偏好设置" > "安全性与隐私"
   - 点击"仍要打开"按钮

### 使用 Homebrew 安装（推荐）

如果你已经安装了 Homebrew，可以使用命令行快速安装：

```bash
# 添加 Claude Code tap
brew tap claude/code

# 安装 Claude Code
brew install claude-code

# 启动应用
claude-code
```

### macOS 常见问题

**问题：无法打开应用**
- 解决方案：在终端运行 `xattr -cr /Applications/Claude\ Code.app`

**问题：命令行工具未找到**
- 解决方案：在 Claude Code 中打开命令面板（Cmd+Shift+P），搜索"Install 'claude' command"

## Linux 安装步骤

### Ubuntu/Debian 系统

使用 APT 包管理器安装：

```bash
# 添加 Claude Code 仓库
curl -fsSL https://code.claude.com/keys/linux.asc | sudo gpg --dearmor -o /usr/share/keyrings/claude-code.gpg

echo "deb [signed-by=/usr/share/keyrings/claude-code.gpg] https://code.claude.com/repos/apt stable main" | sudo tee /etc/apt/sources.list.d/claude-code.list

# 更新包列表
sudo apt update

# 安装 Claude Code
sudo apt install claude-code
```

### Fedora/RHEL 系统

使用 DNF 包管理器安装：

```bash
# 添加仓库
sudo rpm --import https://code.claude.com/keys/linux.asc

sudo tee /etc/yum.repos.d/claude-code.repo <<EOF
[claude-code]
name=Claude Code Repository
baseurl=https://code.claude.com/repos/rpm
enabled=1
gpgcheck=1
gpgkey=https://code.claude.com/keys/linux.asc
EOF

# 安装
sudo dnf install claude-code
```

### Arch Linux

使用 AUR 安装：

```bash
# 使用 yay
yay -S claude-code

# 或使用 paru
paru -S claude-code
```

### 通用 Linux（AppImage）

如果你的发行版不在上述列表中，可以下载 AppImage 版本：

```bash
# 下载 AppImage
wget https://code.claude.com/download/linux/claude-code.AppImage

# 添加执行权限
chmod +x claude-code.AppImage

# 运行
./claude-code.AppImage
```

### Linux 常见问题

**问题：缺少依赖库**
- 解决方案：安装必要的依赖 `sudo apt install libgtk-3-0 libnotify4 libnss3`

**问题：无法启动图形界面**
- 解决方案：确保已安装 X11 或 Wayland 显示服务器

## 首次配置

安装完成后，首次启动 Claude Code 时需要进行一些基本配置：

### 1. 登录账户

1. 启动 Claude Code
2. 点击"登录"按钮
3. 使用你的 Anthropic 账户登录
4. 如果没有账户，点击"注册"创建新账户

### 2. 选择主题

根据个人喜好选择明亮或暗黑主题：
- 点击左下角的设置图标
- 选择"主题" > "颜色主题"
- 选择你喜欢的主题

### 3. 配置快捷键

Claude Code 提供了丰富的快捷键，你可以根据习惯自定义：
- 打开命令面板：`Ctrl+Shift+P`（Windows/Linux）或 `Cmd+Shift+P`（macOS）
- 搜索"键盘快捷键"
- 自定义你常用的快捷键

### 4. 安装扩展（可选）

根据你的编程语言和工作流程，可以安装相应的扩展：
- 点击左侧的扩展图标
- 搜索你需要的扩展（如 Python、JavaScript、Git 等）
- 点击"安装"按钮

## 验证安装

要确认 Claude Code 已正确安装，可以：

1. 打开终端或命令提示符
2. 运行命令：`claude --version`
3. 如果显示版本号，说明安装成功

## 更新 Claude Code

Claude Code 会自动检查更新，当有新版本时会提示你更新。你也可以手动检查更新：

- 打开命令面板（`Ctrl+Shift+P` 或 `Cmd+Shift+P`）
- 搜索"检查更新"
- 如果有更新，点击"立即更新"
