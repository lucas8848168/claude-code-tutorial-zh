# 文件操作和项目管理

Claude Code 提供了强大的文件管理功能，让你能够高效地组织和管理项目文件。本章节将介绍如何使用 Claude Code 的文件操作功能来提高工作效率。

## 文件浏览器基础

### 打开文件浏览器

Claude Code 的文件浏览器位于左侧活动栏。你可以：

1. **点击文件图标**打开文件浏览器
2. **使用快捷键** `Ctrl+Shift+E`（Windows/Linux）或 `Cmd+Shift+E`（macOS）
3. **从菜单**选择"查看" > "资源管理器"

### 文件浏览器的主要功能

- **文件树视图**：显示项目的完整目录结构
- **快速搜索**：在文件浏览器中搜索文件
- **上下文菜单**：右键点击文件或文件夹获取操作选项
- **拖放操作**：拖放文件进行移动或复制
- **文件预览**：点击文件快速预览内容

## 创建和删除文件

### 创建新文件

有多种方式创建新文件：

**方式 1：通过文件浏览器**
1. 在文件浏览器中右键点击文件夹
2. 选择"新建文件"
3. 输入文件名并按 Enter

**方式 2：使用快捷键**
- 按 `Ctrl+N`（Windows/Linux）或 `Cmd+N`（macOS）创建新文件
- 输入文件名和路径

**方式 3：通过命令面板**
- 按 `Ctrl+Shift+P`（Windows/Linux）或 `Cmd+Shift+P`（macOS）
- 搜索"新建文件"
- 输入文件名

### 创建新文件夹

1. 在文件浏览器中右键点击
2. 选择"新建文件夹"
3. 输入文件夹名称

### 删除文件和文件夹

1. 在文件浏览器中右键点击要删除的项目
2. 选择"删除"
3. 确认删除操作

**注意**：删除操作会将文件移到回收站，你可以恢复。

## 文件重命名和移动

### 重命名文件

**方式 1：通过文件浏览器**
1. 右键点击文件
2. 选择"重命名"
3. 输入新名称

**方式 2：快速重命名**
- 选中文件后按 `F2` 键
- 输入新名称

### 移动文件

**方式 1：拖放操作**
- 在文件浏览器中拖动文件到目标文件夹

**方式 2：剪切和粘贴**
1. 右键点击文件，选择"剪切"（或按 `Ctrl+X`）
2. 导航到目标文件夹
3. 右键点击，选择"粘贴"（或按 `Ctrl+V`）

### 复制文件

1. 右键点击文件，选择"复制"（或按 `Ctrl+C`）
2. 导航到目标位置
3. 右键点击，选择"粘贴"（或按 `Ctrl+V`）

## 项目结构最佳实践

### 推荐的项目结构

对于一个典型的 Web 项目，推荐的目录结构如下：

```
my-project/
├── src/                    # 源代码目录
│   ├── components/         # React 组件
│   ├── pages/             # 页面组件
│   ├── utils/             # 工具函数
│   ├── styles/            # 样式文件
│   ├── hooks/             # 自定义 hooks
│   └── App.tsx            # 主应用组件
├── public/                # 静态资源
│   ├── images/
│   ├── fonts/
│   └── index.html
├── tests/                 # 测试文件
│   ├── unit/
│   └── integration/
├── docs/                  # 文档
├── .env                   # 环境变量
├── .gitignore            # Git 忽略文件
├── package.json          # 项目配置
├── tsconfig.json         # TypeScript 配置
└── README.md             # 项目说明
```

### 命名规范

**文件夹命名**
- 使用小写字母和连字符：`my-folder`
- 避免空格和特殊字符
- 使用复数形式表示集合：`components`、`utils`

**文件命名**
- React 组件：PascalCase（`UserProfile.tsx`）
- 工具函数：camelCase（`formatDate.ts`）
- 样式文件：kebab-case（`user-profile.css`）
- 测试文件：`*.test.ts` 或 `*.spec.ts`

### 批量操作

使用 Claude Code 的多选功能进行批量操作：

1. 按住 `Ctrl`（Windows/Linux）或 `Cmd`（macOS）点击多个文件
2. 右键点击选中的文件
3. 选择操作（删除、移动、复制等）

### 搜索和替换文件内容

**全局搜索**
- 按 `Ctrl+Shift+F`（Windows/Linux）或 `Cmd+Shift+F`（macOS）
- 输入搜索词
- 查看所有匹配的文件和位置

**全局替换**
- 在搜索框中输入搜索词
- 点击"替换"按钮
- 输入替换内容
- 选择"全部替换"

## 工作区和多文件夹

### 打开多个文件夹

Claude Code 支持在同一工作区中打开多个文件夹：

1. 点击"文件" > "将文件夹添加到工作区"
2. 选择要添加的文件夹
3. 文件夹会显示在文件浏览器中

### 工作区配置

你可以保存工作区配置：

1. 点击"文件" > "将工作区另存为"
2. 输入工作区名称
3. 下次打开时可以快速恢复相同的文件夹配置

## 文件同步和备份

### 自动保存

Claude Code 支持自动保存功能：

1. 打开设置（`Ctrl+,` 或 `Cmd+,`）
2. 搜索"Auto Save"
3. 选择自动保存模式：
   - `off`：禁用自动保存
   - `afterDelay`：延迟后自动保存（默认 1000ms）
   - `onFocusChange`：切换窗口时自动保存
   - `onWindowChange`：切换应用时自动保存

### 版本控制集成

Claude Code 与 Git 深度集成：

1. 在文件浏览器中查看 Git 状态
2. 使用 Source Control 面板管理版本
3. 直接在编辑器中查看文件变更

## 实用技巧

### 1. 快速打开文件

使用 `Ctrl+P`（Windows/Linux）或 `Cmd+P`（macOS）快速打开文件：

- 输入文件名的一部分
- 支持模糊匹配
- 按 Enter 打开文件

### 2. 最近打开的文件

点击"文件" > "最近打开"查看最近打开的文件列表。

### 3. 文件对比

要比较两个文件的差异：

1. 右键点击第一个文件，选择"选择用于比较"
2. 右键点击第二个文件，选择"与选定的文件进行比较"
3. 查看两个文件的差异

### 4. 文件预览

在文件浏览器中单击文件（不是双击）可以预览文件内容，而不打开编辑标签。

## 常见问题

### Q: 如何恢复已删除的文件？

**A:** 删除的文件会进入回收站。你可以：
1. 打开系统回收站
2. 找到要恢复的文件
3. 右键点击，选择"还原"

### Q: 如何在多个文件中进行查找和替换？

**A:** 使用全局搜索和替换功能：
1. 按 `Ctrl+Shift+F`（Windows/Linux）或 `Cmd+Shift+F`（macOS）
2. 输入搜索词
3. 点击"替换"按钮
4. 输入替换内容
5. 选择"全部替换"

### Q: 如何快速导航到特定文件？

**A:** 使用快速打开功能：
1. 按 `Ctrl+P`（Windows/Linux）或 `Cmd+P`（macOS）
2. 输入文件名
3. 按 Enter 打开
