# 实践练习

在这一章中，你将通过交互式练习和测验来巩固之前学到的知识。

## 练习 1：编写你的第一个函数

让我们从一个简单的练习开始。你需要编写一个函数，接收一个名字作为参数，并返回一个问候语。

```javascript
// 练习 1：编写问候函数
// 要求：编写一个函数，接收一个名字参数，返回格式为 'Hello, [name]!' 的问候语

function greet(name) {
  // 在这里编写你的代码
  return `Hello, ${name}!`;
}

// 测试
console.log(greet("World"));  // 输出: Hello, World!
console.log(greet("Alice"));  // 输出: Hello, Alice!
```

## 练习 2：数组操作

现在让我们练习数组操作。你需要编写一个函数，计算数组中所有数字的总和。

```javascript
// 练习 2：计算数组总和
// 要求：编写一个函数，接收一个数字数组，返回所有数字的总和

function sumArray(arr) {
  // 方法 1：使用 reduce
  return arr.reduce((sum, num) => sum + num, 0);
  
  // 方法 2：使用 for 循环
  // let sum = 0;
  // for (let num of arr) {
  //   sum += num;
  // }
  // return sum;
}

// 测试
console.log(sumArray([1, 2, 3, 4, 5]));  // 输出: 15
console.log(sumArray([10, 20, 30]));     // 输出: 60
```
