## 高级编辑技巧

### 1. 多光标编辑

#### 场景：同时编辑多个位置

```typescript
// 原始代码
const user1 = getUserData(1);
const user2 = getUserData(2);
const user3 = getUserData(3);

// 使用多光标编辑
// 1. 选中第一个 getUserData
// 2. Ctrl+D 选择下一个匹配项
// 3. 继续 Ctrl+D 选择所有匹配项
// 4. 现在可以同时编辑所有匹配项

// 结果
const user1 = fetchUserData(1);
const user2 = fetchUserData(2);
const user3 = fetchUserData(3);
```

### 2. 列选择

#### 场景：编辑多行的特定列

```typescript
// 原始代码
const firstName = 'John';
const lastName = 'Doe';
const email = 'john@example.com';

// 使用列选择
// 1. 点击第一行的 'const'
// 2. Shift+Alt+↓ 向下扩展选择
// 3. 现在可以同时编辑所有行的 'const'

// 结果
let firstName = 'John';
let lastName = 'Doe';
let email = 'john@example.com';
```

### 4. 代码格式化

```typescript
// 格式化快捷键
// Shift+Alt+F / Shift+Option+F - 格式化文档
// Ctrl+K Ctrl+F / Cmd+K Cmd+F - 格式化选中代码

// 示例：格式化前
function foo(){const x=1;return x;}

// 格式化后
function foo() {
  const x = 1;
  return x;
}
```

## 常见问题

### Q: 如何快速导航到特定文件？

**A:** 使用 `Ctrl+P`（Windows/Linux）或 `Cmd+P`（macOS）快速打开文件。

### Q: 如何同时编辑多个位置？

**A:** 使用 `Ctrl+D` 选择下一个匹配项，或使用 `Ctrl+Shift+L` 选择所有匹配项。

### Q: 如何快速格式化代码？

**A:** 使用 `Shift+Alt+F`（Windows/Linux）或 `Shift+Option+F`（macOS）。

### Q: 如何在 Chat 中引用特定代码？

**A:** 选中代码后打开 Chat，选中的代码会自动作为上下文。
